┌─ CARACTERÍTICAS ───────────────────────────────────────────────────────────┐

  LLM:
    • Gemini 1.5 Flash via google-generativeai
    • Temperature = 0.1 (determinístico)

  Memory:
    • ConversationBufferWindowMemory
    • k=5 (mantiene últimos 5 turnos)
    • Permite contexto en conversaciones multiturno

  Pattern:
    • ReAct (Reasoning + Acting)
    • El agente razona qué tool usar automáticamente

  Tools:
    • 3 herramientas personalizadas completamente funcionales
    • Docstrings descriptivos (cruciales para el LLM)
    • Mock data para desarrollo, referencias a GCP para producción
